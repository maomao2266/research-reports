信用市場觀察框架

固收市場信號 → 權益投資決策的系統化映射

🎯 核心命題：信用市場是總體經濟的「內線交易者」。債券投資人比股票投資人更早看到風險 —— 因為他們對下行風險更敏感、對現金流更依賴。系統性觀察固收市場的十個維度，可以提前捕捉經濟週期轉折、信用條件變化、和產業輪動信號。

Layer 1：美國經濟 — 總體週期錨點

📊 核心指標
指標觀察重點擴張信號收縮信號
失業率 薩姆規則（Sahm Rule）：3個月移動均值較前12月低點上升 0.5% = 衰退 < 4.5% > 5% 且上升中
核心 CPI 年增率趨勢。關注「最後一哩」僵固性（住房、服務） 回落至 2-2.5% > 3.5% 且無回落
核心 PCE Fed 首選通膨指標。與 CPI 的差距 = 住房權重差異 接近 2% 目標 > 3%
Fed 通膨目標 點陣圖 + SEP 的 PCE 預測中位數。市場定價 vs Fed 指引的分歧 Fed 轉鴿 Fed 轉鷹
🔑 對台股的意義：美國經濟週期是台股方向性的最大單一驅動因子。出口導向的台灣對美國終端需求高度敏感。失業率上升 → 消費降溫 → 科技供應鏈砍單。核心 CPI 僵固 → Fed 不降息 → 台幣貶值壓力 + 外資流出。

📊 核心指標

短端降 > 長端降		 熊平（bear flattener）
短端升 > 長端升
5Y/30Y 國債利差 曲線斜率。倒掛 = 衰退預期。重新陡峭化 = 復甦初期 陡峭化（steepening）
> 50bps		 倒掛（inversion）

💡 殖利率曲線形態速查：
牛陡（Bull Steepener）：短端利率降得快 → Fed 降息預期升溫 → 利好成長股、科技股
牛平（Bull Flattener）：長端利率降得快 → 衰退擔憂 → 利好防禦型、長天期債券
熊陡（Bear Steepener）：長端利率升得快 → 通膨預期上升 → 利好原物料、銀行
熊平（Bear Flattener）：短端利率升得快 → Fed 升息 → 全體股市承壓

🔑 產業輪動信號：這是整個信用框架中最直接對應「產業選擇」的指標。IG Credit 的 excess return by sector 直接告訴你債券投資人對不同產業的信用看法 —— 這是比股票分析師更早的產業信號。

Industrial 報酬改善 → 景氣循環股轉強
Utility 報酬改善 → 防禦型轉強（避險需求）
Finance 報酬改善 → 銀行、保險轉強（利差擴大 + 信用成本降）

信號整合：從信用市場到權益決策

🎯 三層信號疊加決策框架

Layer 1: 總體週期判斷（美國經濟）
  ├─ 失業率 + 核心 CPI → 四象限定位
  └─ 輸出：現在是 Goldilocks / Overheat / Disinflation / Stagflation？

Layer 2: 信用條件綜合判斷（匯總信用指標）
  ├─ 殖利率曲線形狀 → 利率方向
  ├─ HY spread + fund flows → 風險偏好
  ├─ IG excess return by sector → 產業信用優劣
  ├─ ABS/MBS/CMBS spread → 消費/房貸/商辦健康度
  └─ 輸出：信用在擴張還是收縮？哪些產業信用在改善？

Layer 3: 台股策略映射
  ├─ Goldilocks + 信用擴張 → 全力做多成長股、半導體
  ├─ Overheat + 信用擴張 → 原物料、金融、景氣循環
  ├─ Disinflation + 信用收縮 → 防禦型（公用事業、電信、高股息）
  └─ Stagflation + 信用危機 → 現金為王，等待 HY spread > 800bps 買點

📋 框架定位

信用市場觀察框架是 Top-Down 投資體系的 Layer 1（宏觀）與 Layer 3（產業）之間的橋樑。它不直接選股，但提供兩個關鍵判斷：

1. 現在應該積極還是防禦？（信用條件方向）

2. 哪些產業在信用層面正在改善？（IG excess return by sector）
